Inspections API

Welcome to the Inspections API

This Api will help you manage Inspections by allowing you to have an authentication filter. But also Create, Delete, Edit and Update your Inspections.

API Endpoints

Endpoint Method Description
/api/v1/token POST Provides a JWT (JSON Web Token) authentication after providing username and password
/api/v1/token/refresh POST Allows JWT token refresh for auth purposes and token to last longer to avoid a login every time the user comes to the site
/api/v1/inspections GET Retrieves a list of all inspections.
/api/v1/users/{id} GET Retrieves a specific inspection by its ID.
/api/v1/inspections POST Creates a new inspection.
/api/v1/inspections/{id} PUT Updates an inspection by its ID.
/api/v1/inspections/{id} DELETE Deletes an inspection by its ID.
